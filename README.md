# 快捷键
<Left Alt-h> 光标向左移动
<Left Alt-j> 光标向下移动
<Left Alt-k> 光标向上移动
<Left Alt-l> 光标向右移动
<Left Alt-o> 鼠标左键单击
<Left Alt-a> 鼠标左键双击
<Left Alt-x> 鼠标右键单击
<Left Alt-b> 鼠标滚轮上滑
<Left Alt-n> 鼠标滚轮下滑
# 如何关闭
## 方法一：在powershell终端使用如下命令关闭
```powershell
taskkill /f /im keyboard_to_mouse.exe
```
## 方法二：在任务管理器中找到`keyboard_to_mouse.exe`右键结束任务
